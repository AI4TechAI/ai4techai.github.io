This page allows you to verify the authenticity of certificates issued by AI4Tech. Enter the Credential ID printed on the certificate, or use the direct verification link provided to the participant.

Enter a certificate code above or open this page with a URL like ?id=AI4T-2025-0001.

How this verification works

Each certificate issued by AI4Tech receives a unique Credential ID. When you submit a code, this page checks it against our official registry of certificates and returns the corresponding details, including status and course information.

For transparency, the verification is read-only and publicly accessible. If a certificate is revoked or expires, its status will be updated here.
